# webfollow-app

> RSS reader suppot fever api

- Strive to be the best reader experience
- 致力成为体验最佳的阅读器

## Feature

**feature**

- [x] fever api
- [x] data in local
- [x] ai search (rag)
- [x] recommend
- [x] llm summary + llm filter
- [x] pwa
- [x] List entry ai summary
- [x] List scroll mark as read

**base**

- [x] auto-reader
- [x] text-reader
- [x] podcast-reader
- [x] img-reader
- [x] video-reader
- [x] auto-view
- [x] column-view
- [x] list-view
- [x] card-view
- [x] text-view
- [x] content-view

**todo list**

- [ ] ai search 2.0 (mcp / function-call)
- [ ] export opml
- [ ] data reset
